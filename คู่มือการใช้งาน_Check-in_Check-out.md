## 🎯 ภาพรวมระบบ

ระบบ Check-in / Check-out เป็นระบบสำหรับติดตามการเข้าร่วมกิจกรรมจริงและคำนวณชั่วโมงจิตอาสาจากเวลาจริงที่ทำงาน โดย:

- **Check-in**: บันทึกเวลาที่เริ่มเข้าร่วมกิจกรรม
- **Check-out**: บันทึกเวลาที่เสร็จสิ้นกิจกรรม และคำนวณชั่วโมงจิตอาสาจากเวลาจริง

### ข้อดีของระบบ
- ✅ คำนวณชั่วโมงจากเวลาจริงที่ทำงาน
- ✅ ใช้ QR Code สำหรับความสะดวกและความปลอดภัย
- ✅ Token มีอายุ 5 นาที เพื่อความปลอดภัย
- ✅ ป้องกันการ check-in/check-out ซ้ำ

## ⚠️ เงื่อนไขการใช้งาน

### สำหรับผู้เข้าร่วมกิจกรรม

1. **ต้องสมัครกิจกรรมก่อน**
   - ต้องสมัครกิจกรรมผ่านระบบก่อนจึงจะสามารถ check-in ได้
   - สถานะการสมัคร: `requested`, `confirmed`, หรือ `waitlist`

2. **ต้องเข้าสู่ระบบ (Login)**
   - ต้องเข้าสู่ระบบก่อนสแกน QR Code

3. **ลำดับการใช้งาน**
   - ต้อง **Check-in ก่อน** จึงจะสามารถ **Check-out** ได้
   - ไม่สามารถ Check-out โดยไม่ Check-in

4. **ข้อจำกัด**
   - แต่ละกิจกรรมสามารถ Check-in ได้ **ครั้งเดียวเท่านั้น**
   - แต่ละกิจกรรมสามารถ Check-out ได้ **ครั้งเดียวเท่านั้น**

## 👤 วิธีใช้งานสำหรับผู้เข้าร่วมกิจกรรม

### ขั้นตอนที่ 1: สมัครกิจกรรม

1. เข้าสู่ระบบ
2. ไปที่หน้ารายละเอียดกิจกรรม
3. กดปุ่ม **"สมัครเข้าร่วมกิจกรรม"**
4. รอการอนุมัติจากผู้จัดกิจกรรม (ถ้าจำเป็น)

### ขั้นตอนที่ 2: Check-in (เข้า–กิจกรรม)

เมื่อถึงวันและเวลากิจกรรม:

#### วิธีที่ 1: สแกน QR Code (แนะนำ)

1. **ขอ QR Code Check-in จากผู้จัดกิจกรรม**
   - QR Code จะแสดงในหน้ารายละเอียดกิจกรรม (สำหรับ Admin)
   - หรือผู้จัดกิจกรรมอาจพิมพ์/แสดง QR Code ไว้ที่สถานที่จัดกิจกรรม

2. **เปิดหน้าเว็บสแกน QR Code**
   - ไปที่หน้า `/qr/scan/` หรือ
   - ใช้แอปสแกน QR Code บนมือถือ

3. **สแกน QR Code Check-in**
   - อนุญาตให้แอปใช้กล้อง
   - วาง QR Code ให้อยู่ในกรอบสแกน
   - รอให้ระบบประมวลผล

4. **ตรวจสอบผลลัพธ์**
   - หากสำเร็จ: จะเห็นข้อความ **"Check-in สำเร็จ!"**
   - ระบบจะส่งการแจ้งเตือน (Notification) ให้คุณ
   - บันทึกเวลา Check-in อัตโนมัติ

#### วิธีที่ 2: ใช้ URL โดยตรง

1. **รับ URL หรือ Token จากผู้จัดกิจกรรม**
   - URL ตัวอย่าง: `/check-in/?token=CHECKIN_TOKEN_HERE`

2. **เปิด URL ในเบราว์เซอร์**
   - ต้องเข้าสู่ระบบก่อน
   - ระบบจะประมวลผลอัตโนมัติ

3. **ตรวจสอบผลลัพธ์**
   - หากสำเร็จ: จะเห็นข้อความ **"Check-in สำเร็จ!"**

### ขั้นตอนที่ 3: Check-out (ออก–กิจกรรม)

เมื่อเสร็จสิ้นกิจกรรม:

#### วิธีที่ 1: สแกน QR Code (แนะนำ)

1. **ขอ QR Code Check-out จากผู้จัดกิจกรรม**
   - QR Code Check-out จะแตกต่างจาก QR Code Check-in

2. **เปิดหน้าเว็บสแกน QR Code**
   - ไปที่หน้า `/qr/scan/` หรือ
   - ใช้แอปสแกน QR Code บนมือถือ

3. **สแกน QR Code Check-out**
   - วาง QR Code ให้อยู่ในกรอบสแกน
   - รอให้ระบบประมวลผล

4. **ตรวจสอบผลลัพธ์**
   - หากสำเร็จ: จะเห็นข้อความ **"Check-out สำเร็จ! คุณได้รับ X.XX ชั่วโมงจิตอาสา"**
   - ระบบจะคำนวณชั่วโมงจากเวลาจริง (Check-out - Check-in)
   - ระบบจะส่งการแจ้งเตือนพร้อมจำนวนชั่วโมงที่ได้รับ
   - สถานะการสมัครจะเปลี่ยนเป็น `attended`

#### วิธีที่ 2: ใช้ URL โดยตรง

1. **รับ URL หรือ Token จากผู้จัดกิจกรรม**
   - URL ตัวอย่าง: `/check-out/?token=CHECKOUT_TOKEN_HERE`

2. **เปิด URL ในเบราว์เซอร์**
   - ต้องเข้าสู่ระบบก่อน
   - ระบบจะประมวลผลอัตโนมัติ

3. **ตรวจสอบผลลัพธ์**
   - หากสำเร็จ: จะเห็นข้อความพร้อมจำนวนชั่วโมงที่ได้รับ

### ตรวจสอบสถานะ

- ไปที่หน้า **"โปรไฟล์"** หรือ **"กิจกรรมของฉัน"** เพื่อดู:
  - สถานะการ Check-in/Check-out
  - จำนวนชั่วโมงจิตอาสาที่ได้รับ
  - ประวัติการเข้าร่วมกิจกรรม

## 👨‍💼 วิธีใช้งานสำหรับผู้จัดกิจกรรม (Admin)

### การสร้าง QR Code สำหรับกิจกรรม

1. **ไปที่หน้ารายละเอียดกิจกรรม**
   - URL: `/activity/<id>/`
   - ต้องเป็น Admin หรือผู้จัดกิจกรรม

2. **ดู QR Codes ที่สร้างอัตโนมัติ**
   - ระบบจะสร้าง QR Code 3 แบบ:
     - **QR Code เดิม** (สำหรับ QRScan - ระบบเดิม)
     - **QR Code Check-in** (สำหรับ Check-in)
     - **QR Code Check-out** (สำหรับ Check-out)

3. **ดาวน์โหลดหรือแสดง QR Code**
   - กดปุ่ม **"ดาวน์โหลด QR Code"** เพื่อบันทึกเป็นไฟล์ภาพ
   - หรือแสดงบนหน้าจอ/พิมพ์เพื่อให้ผู้เข้าร่วมสแกน

### การใช้งาน QR Code

#### สำหรับ Check-in

1. **แสดง QR Code Check-in**
   - แสดงที่จุดลงทะเบียน/จุดเริ่มกิจกรรม
   - หรือส่งให้ผู้เข้าร่วมผ่านช่องทางต่างๆ

2. **แนะนำผู้เข้าร่วม**
   - แนะนำให้สแกน QR Code Check-in ก่อนเริ่มกิจกรรม
   - ตรวจสอบว่าแต่ละคน Check-in สำเร็จ

#### สำหรับ Check-out

1. **แสดง QR Code Check-out**
   - แสดงที่จุดสิ้นสุดกิจกรรม
   - หรือส่งให้ผู้เข้าร่วมเมื่อเสร็จสิ้นกิจกรรม

2. **แนะนำผู้เข้าร่วม**
   - แนะนำให้สแกน QR Code Check-out หลังจากเสร็จสิ้นกิจกรรม
   - ตรวจสอบว่าแต่ละคน Check-out สำเร็จ

### ข้อควรระวังสำหรับ Admin

1. **Token มีอายุ 5 นาที**
   - QR Code จะหมดอายุหลังจาก 5 นาที
   - ต้องสร้าง QR Code ใหม่เมื่อหมดอายุ
   - แนะนำให้สร้าง QR Code ใหม่ก่อนเริ่มกิจกรรม

2. **แยก QR Code Check-in และ Check-out**
   - อย่าส่ง QR Code ผิดประเภท
   - Check-in ใช้ QR Code Check-in เท่านั้น
   - Check-out ใช้ QR Code Check-out เท่านั้น

3. **ตรวจสอบการ Check-in/Check-out**
   - ตรวจสอบในระบบว่าใคร Check-in/Check-out แล้วบ้าง
   - ติดตามผู้ที่ยังไม่ได้ Check-in/Check-out

## 📊 การคำนวณชั่วโมงจิตอาสา

### วิธีการคำนวณ

ระบบจะคำนวณชั่วโมงจากเวลาจริงที่ทำงาน:

```
ชั่วโมงที่ได้รับ = เวลา Check-out - เวลา Check-in
```

### ตัวอย่างการคำนวณ

**ตัวอย่างที่ 1: เวลาปกติ**
- Check-in: 09:00 น.
- Check-out: 11:30 น.
- ชั่วโมงที่ได้รับ: **2.50 ชั่วโมง**

**ตัวอย่างที่ 2: เวลาน้อยกว่า 6 นาที**
- Check-in: 09:00 น.
- Check-out: 09:05 น. (5 นาที)
- ชั่วโมงที่ได้รับ: **ใช้ค่า `activity.hours_reward`** (fallback)
  - เพราะเวลาน้อยกว่า 0.1 ชั่วโมง (6 นาที)

### การแสดงผลชั่วโมง

- ชั่วโมงจะแสดงในหน้า **"โปรไฟล์"**
- รวมชั่วโมงจากทั้งระบบเดิม (QRScan) และระบบใหม่ (CheckInOut)
- ชั่วโมงจะถูกบันทึกทันทีหลังจาก Check-out สำเร็จ

## ❓ ปัญหาที่พบบ่อย (FAQ)

### Q1: ทำไมสแกน QR Code แล้วไม่สำเร็จ?

**A:** ตรวจสอบดังนี้:
- ✅ ต้องเข้าสู่ระบบก่อน
- ✅ ต้องสมัครกิจกรรมก่อน
- ✅ QR Code ยังไม่หมดอายุ (มีอายุ 5 นาที)
- ✅ ใช้ QR Code ที่ถูกต้อง (Check-in vs Check-out)
- ✅ กิจกรรมยังไม่ถูกยกเลิก

### Q2: ทำไม Check-out ไม่ได้?

**A:** ตรวจสอบดังนี้:
- ✅ ต้อง Check-in ก่อน
- ✅ ต้องสมัครกิจกรรมก่อน
- ✅ QR Code Check-out ยังไม่หมดอายุ
- ✅ ยังไม่ได้ Check-out ไปแล้ว (สามารถ Check-out ได้ครั้งเดียว)

### Q3: QR Code หมดอายุทำอย่างไร?

**A:**
- QR Code มีอายุ 5 นาที
- ถ้าหมดอายุ: ขอ QR Code ใหม่จากผู้จัดกิจกรรม
- หรือ Admin สามารถสร้าง QR Code ใหม่ได้ที่หน้ารายละเอียดกิจกรรม

### Q4: Check-in/Check-out ซ้ำได้ไหม?

**A:**
- ❌ **ไม่ได้** - แต่ละกิจกรรมสามารถ Check-in ได้ครั้งเดียว
- ❌ **ไม่ได้** - แต่ละกิจกรรมสามารถ Check-out ได้ครั้งเดียว
- หากพยายาม Check-in/Check-out ซ้ำ ระบบจะแจ้งว่าทำไปแล้วเมื่อเวลาใด

### Q5: ชั่วโมงที่ได้รับน้อยกว่าที่คาดหวัง?

**A:**
- ระบบคำนวณจากเวลาจริง (Check-out - Check-in)
- ถ้าเวลาน้อยกว่า 6 นาที จะใช้ค่า `activity.hours_reward` แทน
- ตรวจสอบเวลา Check-in และ Check-out ในระบบ

### Q6: ตรวจสอบสถานะ Check-in/Check-out ได้ที่ไหน?

**A:**
- ไปที่หน้า **"โปรไฟล์"** หรือ **"กิจกรรมของฉัน"**
- ดูประวัติการเข้าร่วมกิจกรรม
- หรือดูในหน้า **"การแจ้งเตือน"** (Notifications)

### Q7: ใช้ QR Code เดิม (QRScan) ยังได้ไหม?

**A:**
- ✅ **ได้** - ระบบเดิมยังทำงานได้ตามปกติ
- แต่แนะนำให้ใช้ระบบ Check-in/Check-out ใหม่เพื่อความแม่นยำในการคำนวณชั่วโมง

### Q8: ต้อง Check-in และ Check-out ทุกกิจกรรมไหม?

**A:**
- ขึ้นอยู่กับกิจกรรม
- บางกิจกรรมอาจใช้ระบบเดิม (QRScan)
- บางกิจกรรมอาจใช้ระบบใหม่ (Check-in/Check-out)
- ตรวจสอบคำแนะนำจากผู้จัดกิจกรรม
